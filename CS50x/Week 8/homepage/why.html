The website uses the following tags:

  • br: For moving to next line
  • header: To make a header
  • h1: To make the largest heading possible
  • a: Adding Links
  • h5: Heading of size 5
  • b: BOLD
  • img: For adding an image
  • div: Divisions
  • meta: For mobile appropriate scaling
  • form: Adding an input box
  • input: For taking input from the user
  • ul: For making an ordered list
  • li: For adding list items.
